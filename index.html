Features

Built for faster UI work

Every feature exists for one reason: to shorten the distance between "I need to check this element" and "got it, moving on."

Live element targeting

Hover over any element to highlight it with measurement crosshairs. Click to lock the selection and pin its full details in the side drawer.

Measurement lines

On-screen crosshair lines show the exact pixel distance from the selected element to all four viewport edges — top, right, bottom, and left — with labelled values.

Color & typography

Background and text colors shown with live swatches. Font family, weight, and size surfaced instantly so you can compare the live page against your design specs.

Box model preview

A browser-style diagram shows the margin, border, padding, and content area for every selected element — the exact view you'd get in DevTools, but right on the page.

One-click copy

Click any property in the drawer to copy its value straight to the clipboard. Speeds up debugging, design-to-dev handoff, and anything else that involves copy-pasting CSS values.

Private by design

No accounts, no servers, no data collection. The extension runs entirely in your browser and never sends anything anywhere.

AI Assistant

Got a question about what a CSS value does, or how to replicate a style? Ask the AI assistant — it knows the extension and frontend development inside out.

How it works

Three steps to inspect anything

Install once, use on every site. No sign-up, no config, no tab-switching.

1

Install the extension

Add UI FACECARD to Chrome in a few seconds — no account required.

2

Toggle the debugger

Click the extension icon and hit "Toggle UI Debugger" to activate the overlay on any page.

3

Click any element

Hover to highlight and measure, click to pin. All details appear in the side drawer — copy what you need and move on.
